# 焚き火 — 考えすぎた心をいったん休ませる場所

**反芻（はんすう）思考のためのセルフケアツール**

同じ考えが頭の中で回り続けて止められないとき、その思考を書き出し、少し整理し、いったん区切るための場所です。

🔗 https://takibi-tools.github.io

## これは何か

「焚き火」は、反芻思考に対するセルフケアツールです。

反芻とは、同じ思考が頭の中で繰り返し回り続け、止めにくくなる状態のこと。考え続けても答えが出ないのに、頭から離れない——その循環に対して「考え続ける」以外の選択肢を提供します。

認知行動療法やマインドフルネスの考え方を参考に、思考の外在化・分類・区切りを段階的に行うフローとして設計しています。

## 設計思想

- 書き出すことで、思考を「自分の中」から「自分の外」に移す
- 分類することで、反芻のパターンに気づく手がかりを作る
- 「抱えたまま閉じる」という選択肢を正当化する
- 解決できない問題を無理に解決しようとしない
- 確認行動を識別する
- 不安を確かめに行く衝動と、自分を整える行動を区別する
- 10件未満では傾向を出さない
- 少ないデータから誤ったパターンを読み取るリスクを避ける
- 傾向分析は「観察」として提示し、「診断」としては扱わない

## セルフケアツールとしての安全設計

このツールは、AIとの協働で設計・実装されています。その過程で、以下の安全設計上の判断を行いました。

- 感情をスコアリングしない
- AIが「あなたは大丈夫」と保証しない
- ユーザーの記録を外部に送信しない
- 記録はすべてブラウザ内の `localStorage` に保存する
- 「専門家に相談してください」を免責ではなく、選択肢として提示する
- 確認行動を識別し、実行を保留する設計を組み込む
